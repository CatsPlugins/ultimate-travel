== Description ==
 
In short, this is specialized for travel agency websites. You will be able to manage the tours, hotels, online booking and receive the reservation via email.

* Online Demo
* Backend Demo
* Documentation

SAVE YOUR TIME - SPEED UP DEVELOPMENT PROCESS

Ultimate Travel – Premium WordPress Travel Plugin allow you to plug and play immediately after the installation.

**In the Backend** you will be able to manage:

- Tour: Tours detail, tour attributes (such as the additional services for tours), tour pricing and online booking, tour destinations, and regions.

- Transportation service: car, vehicle, motorbike for hire. All the transportation come along with models, trademark, produced year.

- Staying service: hotel with rooms, additional service, price, contact information, gallery, featured image....

- Booking management: you will be able to manage all the information of reservation

**On the Frontend**, we have built the pages that you just need to use, no need to config at all.

- Tour detail
- Hotel detail
- Car detail
- Tour/Hotel/Car archive
- Booking form

**Widgets**: Tour List, Tour Filter, Car Filter
**Shortcode**: Tour by region, tour by tag

== Installation ==
 
This section describes how to install the plugin and get it working.
 
e.g.
 
1. Upload `ultimate-travel` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
